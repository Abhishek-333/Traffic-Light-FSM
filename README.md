# Traffic-Light-FSM

Moore FSM for a simple traffic-light controller.

Files:
- `traffic_fsm.v` - FSM RTL
- `traffic_fsm_tb.v` - testbench

Simulate to verify sequence: RED -> NS_GREEN -> NS_YELLOW -> RED.
